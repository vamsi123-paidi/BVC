🔤 1. Strings (Deep Understanding)

📖 Definition

A string is a sequence of characters enclosed in quotes

python
name = "Python"


---

🧠 Key Characteristics

* Ordered → index-based
* Immutable → cannot change directly
* Supports slicing

---

🔍 Access Characters

python
text = "Python"

print(text[0])   P
print(text[-1])  n


---

🧠 Important Insight

👉 Strings behave like lists (but cannot be modified)

---

🧪 Practice Tasks

1. Print first and last character
2. Print middle character
3. Try modifying a character → observe error

---

✂️ 2. String Slicing

📖 Definition

Extracting part of a string

---

Syntax

python
text = "Python"

print(text[0:4])   Pyth
print(text[::2])   Pto


---

🧠 Concept

* Start included
* End excluded
* Step optional

---

🧪 Practice Tasks

1. Extract first 3 characters
2. Reverse string
3. Skip characters using step

⚙️ 3. String Methods (Very Important)

📖 Definition

Built-in functions to manipulate strings

---

🔹 Common Methods

python
text = "hello world"

print(text.upper())     HELLO WORLD
print(text.lower())     hello world
print(text.title())     Hello World
print(text.strip())     remove spaces
print(text.replace("hello", "hi"))


---

🔍 Split & Join

python

print(text.split(","))   ['a', 'b', 'c']


---

🧠 Real Use

* Cleaning user input

* Processing text

🔄 6. Advanced Loops

---

🔹 Break Statement

python
for i in range(10):
    if i == 5:
        break
    print(i)


👉 Stops loop completely

---

🔹 Continue Statement

python
for i in range(5):
    if i == 2:
        continue
    print(i)


👉 Skips current iteration

---

🔹 Pass Statement

python
for i in range(5):
    pass


👉 Placeholder (does nothing)

---

🧠 Real Use

* Filtering data
* Skipping unwanted values

---

🧪 Practice Tasks

1. Print numbers except 3
2. Stop loop at 7
3. Skip even numbers

---

🔢 7. Nested Loops

📖 Definition

Loop inside another loop

---

Example

python
for i in range(3):
    for j in range(3):
        print(i, j)


---

🧠 Real Use

* Matrix operations
* Pattern printing

---

🧪 Practice Tasks

1. Print pairs (i, j)
2. Print multiplication table

⭐ 8. Pattern Programs (Important for Logic)

---

🔹 Square Pattern

python
for i in range(5):
    print("* " * 5)


---

🔹 Right Triangle

python
for i in range(5):
    print("* " * (i + 1))


---

🔹 Reverse Triangle

python
for i in range(5):
    print("* " * (5 - i))


---

🧠 Why Patterns?

* Improve logic
* Understand loops deeply

---

🧪 Practice Tasks

1. Pyramid pattern
2. Number pattern
3. Reverse pyramid
