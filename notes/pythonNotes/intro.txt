🌍 1. Introduction to Python

📖 What is Python?

Python is a high-level, interpreted programming language that allows humans to write instructions in a simple, readable format.

👉 In simple terms:
Python lets you communicate with the computer using almost English-like sentences

---

🧠 Why Python was Created

Earlier languages (like C, Java) were:

* Complex to write
* Hard to understand
* Required more lines of code

Python was created to solve this problem:

* Less code
* More readability
* Faster development

---

👤 Creator

* Developed by Guido van Rossum in 1991
* Goal: Make programming simple for everyone

---

⚡ Where Python is Used (Real Understanding)

* Automation → Rename files, send emails
* Web apps → Backend logic
* Data → Analysis & visualization
* AI → Machine learning models

👉 Real-life example:

* Automatically sorting files on your laptop
* Creating a bot that replies to messages

⚙️ 2. How Python Works

📖 Execution Flow

1. Write code (`.py` file)
2. Python Interpreter reads it
3. Converts into bytecode
4. Executes line-by-line

---

🧠 Key Concept

👉 Python runs line by line (top → bottom)

This means:

* Order of code matters
* Errors stop execution

---

🧱 4. Variables

📖 Definition

A variable is a container used to store data values

---

🧠 Deep Understanding

Think of variable like a box with a label

python
age = 20


* `age` → label
* `20` → value stored

---

⚡ Key Features

* No declaration keyword
* Type is automatic
* Value can change

python
x = 10
x = 20


---

⚠️ Rules

* Cannot start with number
* No spaces
* Case-sensitive
