🔁 1. Iterators

📖 Definition

An iterator is an object that allows you to traverse (loop through) elements one by one

---

🧠 Core Concept

* Uses:

  * `__iter__()` → initializes iterator
  * `__next__()` → gives next value

---

📌 Example

python
arr = [10, 20, 30]

it = iter(arr)

print(next(it))
print(next(it))
print(next(it))


---

📤 Output


10
20
30


---

🧠 Important Insight

👉 `for loop` internally uses iterators

---

🌍 Real-World Use

* Reading large files line by line
* Streaming data

---

🧪 Practice Tasks

1. Create iterator from list
2. Use `next()` manually
3. Try one extra `next()` → observe error

⚡ 2. Generators

---

📖 Definition

Generators are functions that yield values one at a time instead of returning all at once

---

🧠 Why Generators?

* Save memory
* Efficient for large data

---

📌 Example

python
def count(n):
    for i in range(n):
        yield i

g = count(3)

print(next(g))
print(next(g))
print(next(g))


---

📤 Output


0
1
2


---

🧠 Difference

| Return        | Yield           |
| ------------- | --------------- |
| Ends function | Pauses function |
| Returns all   | One by one      |

---

🌍 Real-World Use

* Large datasets
* Pagination
* File processing

---

🧪 Practice Tasks

1. Create generator for squares
2. Print values using loop
3. Compare with list

🎭 3. Decorators

---

📖 Definition

A decorator is a function that modifies another function without changing its code

---

🧠 Core Idea

👉 “Wrap a function with extra behavior”

---

📌 Example

python
def my_decorator(func):
    def wrapper():
        print("Before")
        func()
        print("After")
    return wrapper

@my_decorator
def say_hello():
    print("Hello")

say_hello()


---

📤 Output


Before
Hello
After


---

🌍 Real-World Use

* Logging
* Authentication
* Performance tracking

---

🧪 Practice Tasks

1. Create decorator for logging
2. Add message before function runs
3. Wrap multiple functions
