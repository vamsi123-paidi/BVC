⚡ 1. What is Asynchronous Programming?

📖 Definition

Asynchronous programming allows a program to run multiple tasks without waiting for each one to finish

---

🧠 Simple Understanding

❌ Synchronous (Normal)

* Task 1 → wait → Task 2 → wait → Task 3

✅ Asynchronous

* Task 1 starts
* Task 2 starts
* Task 3 starts
  👉 All run together

---

🌍 Real-Life Example

* Downloading files
* Loading multiple APIs
* Chat apps

---

🧪 Practice Thinking

👉 Opening 3 websites at once vs one-by-one

---

🔑 2. Core Keywords

| Keyword   | Meaning                         |
| --------- | ------------------------------- |
| `async`   | Defines async function          |
| `await`   | Waits for task without blocking |
| `asyncio` | Python async module             |

---

📌 3. Basic Example

python id="0p7d5c"
import asyncio

async def hello():
    print("Start")
    await asyncio.sleep(2)
    print("End")

asyncio.run(hello())


---

📤 Output

text
Start
(2 sec delay)
End


---

🧠 Explanation

* `async def` → async function
* `await` → pause without blocking
* `sleep()` simulates delay

🔥 4. Multiple Tasks (Concurrency)

python id="55p0x8"
import asyncio

async def task(name):
    print(f"{name} started")
    await asyncio.sleep(2)
    print(f"{name} finished")

async def main():
    await asyncio.gather(
        task("Task 1"),
        task("Task 2"),
        task("Task 3")
    )

asyncio.run(main())


---

📤 Output

text
Task 1 started
Task 2 started
Task 3 started
(2 sec delay)
Task 1 finished
Task 2 finished
Task 3 finished


---

🧠 Insight

👉 All tasks run together → faster execution

📌 Example: Fetch API Data

python id="4r44q0"
import asyncio
import aiohttp

async def fetch_data(url):
    async with aiohttp.ClientSession() as session:
        async with session.get(url) as response:
            data = await response.json()
            print(data)

asyncio.run(fetch_data("https://jsonplaceholder.typicode.com/todos/1"))


---

📤 Output (example)

text
{'userId': 1, 'id': 1, 'title': '...', 'completed': False}


---

🧠 Explanation

* `ClientSession()` → opens connection
* `await response.json()` → async data fetch
* Non-blocking request

🔥 6. Multiple API Calls (Real Power)

python id="7xv4ru"
import asyncio
import aiohttp

urls = [
    "https://jsonplaceholder.typicode.com/todos/1",
    "https://jsonplaceholder.typicode.com/todos/2",
    "https://jsonplaceholder.typicode.com/todos/3"
]

async def fetch(session, url):
    async with session.get(url) as res:
        data = await res.json()
        print(data["id"], data["title"])

async def main():
    async with aiohttp.ClientSession() as session:
        tasks = [fetch(session, url) for url in urls]
        await asyncio.gather(*tasks)

asyncio.run(main())


---

📤 Output (example)

text
1 Task title...
2 Task title...
3 Task title...


---

🧠 Key Insight

👉 All APIs fetched at the same time

⚖️ 7. Sync vs Async (Comparison)

| Feature   | Sync         | Async    |
| --------- | ------------ | -------- |
| Execution | One by one   | Parallel |
| Speed     | Slow         | Fast     |
| Use case  | Simple tasks | APIs, IO |
